# Contributing

We welcome contributions to this codebase. Please file issues or PRs in this github repo if there are changes you would like to see. We have no Contributors License Agreement and all accepted code will be released under the Apache 2.0 License.

Sensitive security-related issues should be reported to security@confident.security.

Note: this code syncs with our monorepo and we use a tool called [copybara](https://copybara.io) to sync changes here. Nothing in your use of the repo should change except that when a PR is accepted we will merge it in our repo and the one here will be closed when the changes are synced over.

We expect all contributors to follow our (Code of Conduct)[https://www.contributor-covenant.org/version/3/0/]. If you have an issue to report, please email support@confident.security.
